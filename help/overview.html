Manual Sec Builder

Tool for building and editing routes. Saves and loads routes in .sec,.rte and .sr format.

Loading saved routes

Use the load menu to load routes. Loading a route will discard the current route.

Inserting Sections and Dummys into route

New sections/dummys can be inserted into the route through the insert menu. They will be inserted into the route at the row in the 'insert at row' spinbox.

They can be inserted:

  • From .sec file (value in reversed? dropdown will be used for direction)
  • From .rte file (direction will be found using direction field of feature with section label)
  • From .sr file (using direction in file)
  • From selected feature of layer(value in reversed? dropdown will be used for direction)

    • Routes table

    The routes table contains every section in the route(ordered consecutively) and whether it is in the forward or reversed direction.

    Right clicking the route table will produce a menu with options to:

  • Delete selected rows
  • Delete all rows
  • Select on layer(selects features of layer that are selected in routes table)
  • Select from layer(selects rows of route table that are selected on layer)

    • Rows can be rearranged by dragging and dropping
